CGPerfisDeVegetacao-Tester

1) Introducao
Progama para visualizacao de perfis de vegetacao.
Simples e facil gera uma saida do tipo svg(Obs pode ser lido na maioria dos navegadores).

2)Requerimentos minimos
Ter java runtime envoriment 1.6 instalada e configurada corretamente.
Ter 1MB de memoria ram. Funciona com maquinas com memoria inferior, porem é altamentente
recomendavel que o para este caso nao explore entradas com muitas celulas e modelos.
So: independente desde de que tenha a maquina java.

3)Instalacao e execucao 
O programa nao necessita instalar, basta abrir o arquivo "cgPerfis.java" pelo explorer ou usar no prompt de 
comando dentro da pasta que se encontra o programa, a linha de comando: java -jar cgPerfis.jar

4)Duvidas e questoes

-O programa nao abre algumas janelas que tenho como opcoes.
O programa se encontra na fase tester, por isso tem algumas funcoes limitadas.

-O programa demora muito a processar o arquivo
Reabra o programa.
